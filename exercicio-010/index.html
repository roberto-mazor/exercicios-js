Exercício 010

10 Calcular a média de N alunos.
Vamos fazer um programa para calcular a média de um aluno. Ao apresentar o resultado, o programa deve perguntar se gostaria de calcular a média de outro aluno, caso a resposta seja SIM, o programa deve executar novamente. Declare as variáveis que irão armazenar o nome, as notas e a média do aluno.
