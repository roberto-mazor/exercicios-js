Exercício 009

009 Exibindo uma mensagem várias vezes.
A missão consiste em repetir uma mesma frase uma certa quantidade de vezes. Nós já fizemos isso no exemplo sobre laço de repetição, para essa missão vamos perguntar para o usuário quantas vezes ele quer repetir a mensagem. Vamos criar uma variável para ser nosso contador e perguntar quantas vezes ele deseja repetir a mensagem.
